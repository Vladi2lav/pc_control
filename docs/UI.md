# UI.py — Главное окно приложения

Полупрозрачный frameless dark-glass блок с 4 режимами меню,
боковой навигацией, магнитом к краям экрана и кастомным titlebar.

---

## Архитектура окна

```
GlassWindow (QWidget, frameless, translucent)
│
├── _TitleBar            — кастомная панель заголовка
│   ├── Drag-area        — за него тащим окно
│   ├── Лого + заголовок
│   ├── _MenuModeButton  — переключатель режима меню (4 состояния)
│   ├── Кнопка свернуть  — _
│   └── Кнопка закрыть  — ✕
│
├── _body (HBox)
│   ├── _SideBar         — боковое меню (иконки-квадратики)
│   │   └── _ModuleButton× N — квадратик с иконкой и тултипом
│   └── _content         — область содержимого модулей
│
└── Snap-логика          — магнит к краям экрана при отпускании
```

---

## 4 режима меню (MenuMode)

| Режим | Значение | Описание |
|-------|----------|----------|
| `STANDARD` | 1 | Обычное окно по центру/в любом месте экрана |
| `COMPACT`  | 2 | Компактное окно (меньший размер) по центру |
| `SIDE_THIN` | 3 | Прижато к краю, боковая панель закрыта (иконки) |
| `SIDE_OPEN` | 4 | Прижато к краю, боковая панель открыта (иконки + названия) |

Переключение через кнопку в тайтлбаре или через snap при перетаскивании.

---

## Snap (магнит к краям)

При отпускании окна:
- если окно ближе 40px к **левому** краю → прижимается к левому краю
- если окно ближе 40px к **правому** краю → прижимается к правому краю

При прижатии к краю автоматически активируется режим `SIDE_THIN` (боковая панель).
При откреплении возвращается к `STANDARD`.

---

## Боковая панель (_SideBar)

В режимах `SIDE_THIN` и `SIDE_OPEN` — вертикальная панель с квадратными кнопками модулей.

- **SIDE_THIN**: только иконки, при наведении показывается имя (tooltip).
- **SIDE_OPEN**: иконки + текстовые подписи под каждой кнопкой.
- **Авто-раскрытие**: при наведении на сайдбар в режиме `SIDE_THIN` он автоматически переключается в `SIDE_OPEN`.

При клике на кнопку модуля — открывается всплывающее окно (popup) этого модуля рядом с боковой панелью.

---

## Titlebar (_TitleBar)

- Перемещение окна — **только за titlebar** (drag_pos вычисляется только если клик попал в titlebar)
- Кнопка **свернуть (—)** — `showMinimized()`
- Кнопка **закрыть (✕)** — плавный fade-out + остановка core

---

## Подключение модулей

```python
# В GlassWindow.__init__ или run():
from modules.db_explorer import DBExplorerModule
self._mount(DBExplorerModule, icon="🗄", name="DB Explorer")

# _mount добавляет модуль в боковую панель и content area
```

Каждый модуль должен реализовывать:
- `__init__(core, parent)` — конструктор
- `.widget` → `QWidget` для вставки
- `.safe_close()` → корректное завершение

---

## Edge snap + mode transitions

```
Drag → mouse release
    ├── dist_left < 40px  → snap LEFT  → mode = SIDE_THIN
    ├── dist_right < 40px → snap RIGHT → mode = SIDE_THIN
    └── else              → mode = STANDARD (если был SIDE_*)
```
