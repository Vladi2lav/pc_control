# DB Explorer Module

Модуль для просмотра и управления базой данных SQLite. 
Поддерживает просмотр таблиц, выполнение SQL-запросов и тестирование функций ядра.

---

## Возможности

1. **Таблицы**:
   - `records`: Просмотр основных записей.
   - `kv_store`: Просмотр хранилища ключ-значение.
   - `event_log`: Просмотр лога событий.
2. **SQL Терминал**:
   - Выполнение произвольных `SELECT` запросов.
   - Вывод результатов в текстовое поле с подсветкой.
3. **Инструменты**:
   - Добавление тестовых записей.
   - Управление KV-хранилищем (Set, Get, Del).
   - Логирование тестовых событий.
   - Очистка данных.

---

## Технические детали

- **Связь с ядром**: Все запросы выполняются асинхронно через `core.submit()`.
- **UI**: Построен на основе `UIBase` (XML-рендеринг).
- **Стили**: Кастомные QSS-стили для вкладок, кнопок и таблиц.

---

## Использование в UI

```python
from modules.db_explorer import DBExplorerModule
self._mount(DBExplorerModule, icon="🗄", name="DB Explorer")
```

Модуль автоматически создает свои вкладки и подключает логику к XML-элементам.

---

## Структура файлов

- `explorer.py`: Логика модуля и описание UI-классов.
- `layout.xml`: Структура интерфейса.
- `__init__.py`: Экспорт основного класса модуля.
